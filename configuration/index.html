Configuration

Bee uses the Toml standard as a configuration file. If you are unsure about syntax, you can have a look at the official Toml specifications.

The default configuration file is config.toml. You can change the configuration file's path or name by using the -c or --config flag.

For example: 

bee -c config_example.toml
